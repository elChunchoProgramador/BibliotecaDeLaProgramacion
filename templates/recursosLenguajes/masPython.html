    Para instalar python primero tenemos que descargarlo del sitio Oficial, cuando ya lo tengas descargado procede a ejecutar el instalador. IMPORTANTE: si tienes un sistema operativo de 32 bits descarga el exe correspondiente, si tienes un sistema operativo de 64 bits descarga el correspondiente.

  • muy importante selecciona la casilla Add Python 3.x to PATH
  • dale click a install Now
  • permite que esta aplicacion haga cambios al dispositivo
  • close

    • Ahora verificaremos si se instalo correctamente.


  • abre una consola de comandos(cmd)
  • escribe python y dale enter
  • deberia aparecerte lo siguiente: Python 3.x.x ...
